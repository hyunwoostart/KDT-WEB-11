## 1.28

- 구조분해 할당
- 배열이나 객체의 속성을 해체해 그 값을 개별변수에 담는것

### 배열 구조 분해 할당

const [변수] = 배열;
- 각 변수에 배열의 인덱스 순으로 값 대응
- 구조분해 시 변수의 값이 undefined 일 때 기본값 할당 가능
- 구조분해 없이 두 변수의 값 교환도 가능

### 객체 구조 분해 할당
const { 변수 } = 객체;
- 객체 안의 속성을 변수명으로 사용
- 콜론(:)이용햇 새 변수명을 선언하고, 원래의 값을 새 변수명에 할당 가능
